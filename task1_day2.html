Difference between Document and Window objects

Feb 23, 2024


Binary Object Model

The current browser window or tab is represented by objects in the Browser Object Model. The window object, which can occasionally represent an iframe or a window or tab, is the highest item in the BOM. Properties of a window object include the device's screen, location, and browsing history, among others.

While a window object in a multi-tab browser represents a single tab, its methods (such as resizeTo()) and properties (such as innerHeight and innerWidth) impact the entire browser window.


Data Object Model

A web page's Document Object Model is created by the browser when it loads. The HTML document is represented as a tree of objects (HTML, HEAD, BODY, and other HTML tags) by the document object. The HTML document is represented by this root element.

Document vs Window

A document object is created for each HTML document that loads into a window. The page's contents are contained in the document. JavaScript may add, remove, and change the HTML elements, CSS styles, and properties on a page by using the document object.


a. In the JavaScript object hierarchy, the window is located at the root or top level. It is the root object of the Document object model (DOM) and a global/root object in JavaScript. An object inside the window object is called a document, and we manipulate objects inside documents using document objects.


b. The window object is a representation of a tab or window that holds a DOM document. Document object is the property of window object which references the DOM document loaded in that window.
A document object can be accessed either by using window.document property or using document object directly, as window is global object. But windows objects properties cannot be accessed inside the document.


c. Another difference is that window and document objects both contain properties and methods. The properties related to it are stored in the document object. While the properties related to it are stored in the window object. A few method names are the same in both objects, but their actions differ. For example, document.open() generates a blank document inside the window while window.open() starts a new tab or window.


d. Document is an object of window. Window is an object of the browser.
